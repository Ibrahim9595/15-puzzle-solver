# 15 Puzzle Solver:

This simple app generate a random 15 puzzle and solve it using a simple heuristic search (Manhaten distance as heuristic)

# Use The app:
1. `git clone https://github.com/Ibrahim9595/15-puzzle-solver.git`
2. `cd 15-puzzle-solver`
3. `npm install`
4. `npm start`
5. `open the browser to http://localhost:8000`
